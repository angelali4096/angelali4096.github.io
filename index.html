Keva Block Simulation Using Blendr

Blender is an open source 3D modeling and animation software that uses the Bullet physics engine to simulate collions. Our goal is to simulate the orientation of each plank at each timestep. It is much easier to build a tower visually in a simulator, like Blender, and have its backend report back the orientations rather than building a tower from raw OpenGL code and doing numerical differentiation manually.

Build the tower requires knowledge of the Blender interface. Since the interface changes drastically between versions, I will be referring to the 2.79 version.
